## Code blocks

A language-neutral code block:

```
func:
    preformatted text
```

A code block in Python:

```python
def func(n: int) -> str:
    return str(n)
```

A code block in Java:

```java
class Simple {
    public static void main(String args[]) {
        System.out.println("Hello Java!");
    }
}
```

A code block in JSON:

```json
{
    "boolean": true,
    "integer": 42,
    "string": "value",
    "list": [1,2,3]
}
```
